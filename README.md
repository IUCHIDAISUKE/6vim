# 6vim

I've developed simpleTextEditor called 6vim.  
Thank you for sharing [Build Your Own Text Editor](https://viewsourcecode.org/snaptoken/kilo/index.html). :)

## Mac key

- `Home`
  - `fn` + ←
- `End`
  - `fn` + →
- `Page Up`
  - `fn` + ↑
- `Page Down`
  - `fn` + ↓
